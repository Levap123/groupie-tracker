
# Groupie Tracker

This project about musicians!

## Run Locally

Clone the project

```bash
  git clone git@git.01.alem.school:LevapMik/groupie-tracker.git
```

Go to the project directory

```bash
  cd groupie-tracker
```

Install dependencies

```bash
  npm i --prefix front-end
```

Start the server

```bash
  make run
```

go to http://localhost:3000
